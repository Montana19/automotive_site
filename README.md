# Landing Page — Верстка по шаблону из Photoshop

Одностраничный промо-сайт, свёрстанный по дизайн-макету из Photoshop.

## 🔍 Особенности

- Полное соответствие макету
- Семантическая вёрстка HTML5
- Чистый CSS без фреймворков
- Корректное отображение во всех современных браузерах

## 🛠 Технологии

- HTML5
- CSS3
- Photoshop (чтение макета)
- Адаптивная / responsive верстка
