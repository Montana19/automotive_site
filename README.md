# Landing Page — Pixel Perfect верстка по Figma-макету

Одностраничный промо-сайт, свёрстанный по дизайн-макету из Figma с соблюдением точности до пикселя.

## 🔍 Особенности

- Полное соответствие Figma-макету (Pixel Perfect)
- Семантическая вёрстка HTML5
- Чистый CSS без фреймворков
- Корректное отображение во всех современных браузерах

## 🛠 Технологии

- HTML5
- CSS3
- Figma (чтение макета)
